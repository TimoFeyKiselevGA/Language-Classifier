Welcome to Language Classifier! Language Classifier is my school project. Its purpose is to classify different languages in text. Right now the project supports 12 languages. Those languages are: Czech, Russian, French, German, English, Polish, Italian, Ukrainian, Finnish, Bulgarian, Spanish and Portuguese. To use it, you must paste your text into the input field at the home page and click the "Detect Language" button. Formatting and text written between "<" and ">" will be removed. Empty spaces and new lines too. But other special characters should stay intact. Remember that input text should be atleast 30 words long for classifier to work well, and you should use only supported languages. By clicking Documentation button you can learn more information about the project, but right now only czech documentation is avalible.
